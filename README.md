# Portfolio Website

A modern, responsive portfolio website showcasing professional experience, skills, and projects as a Software Engineer, Cloud & DevOps Specialist, and GIS & AI Enthusiast.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works on all devices (mobile, tablet, desktop)
- **Dark/Light Mode**: Toggle between dark and light themes with preference saving
- **Single Page Application**: Smooth navigation between sections (Home, About, Portfolio, Blogs, Contact)
- **SEO Optimized**: Includes meta tags, Open Graph, and Twitter Card support
- **Accessible**: ARIA labels, semantic HTML, and keyboard navigation support
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3/SCSS**: Styling with CSS variables and SCSS
- **Vanilla JavaScript**: ES6+ JavaScript (no frameworks)
- **Font Awesome**: Icons
- **Google Fonts**: Poppins font family

## 🎨 Sections

1. **Home**: Introduction with professional summary and key expertise areas
2. **About**: Detailed skills breakdown, work experience timeline, and education
3. **Portfolio**: Showcase of projects with links to GitHub and demos
4. **Blogs**: Technical blog posts and articles
5. **Contact**: Contact information and email form

## 🔧 Setup & Usage

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process required - works directly in the browser

### Development (Optional)

If you want to modify SCSS files:

1. Install Sass compiler:
   ```bash
   npm install -g sass
   ```

2. Watch for changes:
   ```bash
   sass --watch styles/styles.scss:styles/styles.css
   ```
